# Customer Churn Prediction

## 📌 Project Overview

This project aims to predict customer churn using machine learning techniques.
Customer churn refers to customers who stop using a company's service.

---

## 📊 Dataset

* Telco Customer Churn Dataset
* Contains customer details like:

  * Gender
  * Tenure
  * Monthly Charges
  * Contract Type
  * Churn status

---

## ⚙️ Steps Performed

### 1. Data Loading

* Loaded dataset using Pandas

### 2. Data Cleaning

* Converted `TotalCharges` to numeric
* Handled missing values

### 3. Exploratory Data Analysis (EDA)

* Churn distribution
* Monthly charges analysis
* Data visualization using Seaborn & Matplotlib

### 4. Data Preprocessing

* Converted categorical variables using encoding

### 5. Model Building

* Used Logistic Regression model

### 6. Model Evaluation

* Evaluated using accuracy score and classification report

---

## 📈 Result

* Model successfully predicts customer churn
* Achieved good accuracy on test data
